# Sales Prediction using Machine Learning (Regression)

## Overview
This project focuses on predicting sales using machine learning regression techniques based on historical data.

This project was completed under the guidance of Mr. Kushal Sharma (EY Trainer & Guest Lecturer).

## Objectives
- Build a regression model to predict sales  
- Analyze relationships between input features and sales  
- Improve model performance through preprocessing  
- Evaluate prediction accuracy  

## Methodology
- Data cleaning and preprocessing  
- Feature selection and preparation  
- Model training using regression techniques  
- Evaluation using performance metrics  

## Key Insights
- Certain features showed a strong influence on sales predictions  
- Model performance improved after proper preprocessing  
- Regression models can effectively estimate future sales trends  

## Tools Used
- Python (Pandas, Scikit-learn, Matplotlib)

## Project Structure
- notebook/: Jupyter notebook containing full model implementation  

## Learnings
- Data preprocessing significantly impacts model accuracy  
- Feature selection plays a key role in predictions  
- Model evaluation is essential for reliable results  

## Conclusion
This project demonstrates how machine learning can be used to predict sales and support data-driven decision-making
